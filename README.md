## Installation
```
$ pip install git+https://github.com/JoMingyu/opgg-helper.git
```

## Usage
```
$ opgg 미드 이렐리아
$ opgg mid irelia
$ opgg 미드 이렐
$ opgg 이렐 미드
$ opgg 이렐 ㅁㄷ
$ opgg ㅅㅍ 파이크
$ opgg 자야 .
# -> The position with the highest selection percentage for the champion is automatically selected
```
